# 💰 财务自由模拟评估系统 v1.0

> 输入家庭资产、收入、支出、小孩抚养等信息，逐年模拟现金流直到 85 岁，判断是否实现财务自由。

---

## 🚀 快速开始

### 方式一：打包程序（推荐，无需安装 Python）

将 `dist/财务自由评估/` 文件夹拷贝到任意目录，双击 `财务自由评估.exe` 即可运行。

### 方式二：源码运行（需 Python 3.10+）

```bash
pip install -r requirements.txt
streamlit run app.py
```

或直接双击 `启动.bat`。

---

## 📋 功能概览

| 模块 | 功能 |
|------|------|
| 输入 | 评估对象（个人/夫妻）、7 类资产、3 类负债、4 类收入、月消费、小孩教育（学前/小学/中学/大学 4 阶段）、意外预留、经济参数 |
| 计算 | 逐年模拟现金流，通胀调整，退休归零，资产收益率复利，小孩教育费按阶段自动匹配 |
| 显示 | 财务自由判定结果、净资产趋势折线图、年度收支柱状图、年度明细数据表 |
| 保存 | 参数下载/加载（JSON）、结果下载/加载（JSON）、明细导出（CSV/Excel）、参数自动持久化 |

## 📦 重新打包

修改源码后：

```bash
python -m PyInstaller --clean --noconfirm FinFreedom.spec
```

或双击 `重打包.bat`。

## ⚙️ 默认参数

| 类别 | 默认值 |
|------|--------|
| 年龄 | 本人 35、配偶 35、退休 60、预期寿命 85 |
| 总资产 | 存款 50 万 + 现金 5 万 + 基金 5 万 + 股票 10 万 + 房产 300 万 + 车辆 20 万 + 其他 10 万 = **400 万** |
| 负债 | 全 0 |
| 年收入 | 分红 **15 万** |
| 月消费 | **15,000** |
| 小孩 | 1 个，学前 0-5 岁 5 万/年 + 小学 6 年 10 万/年 + 中学 6 年 10 万/年 + 大学 7 年 10 万/年，赞助 20 万 |
| 意外预留 | **10 万**（一次性扣除） |
| 通胀率 / 收益率 | 均为 **2%** |

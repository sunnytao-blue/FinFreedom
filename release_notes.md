## v1.1

- **日期**：2026-06-08
- **作者**：Sunnytao

### 变更

- 房产估值默认值 300万 → **200万**
- 意外预留默认值 10万 → **20万**，分摊方式 一次性扣除 → **逐年均摊**
- 月生活消费默认值 15,000 → **10,000**
- 新增**房贷月供**输入（默认 3,000元/月，还款期 20 年）
- 新增**车贷月供**输入（默认 2,000元/月，还款期 3 年）
- 年度明细表新增房贷支出、车贷支出列（含 CSV/Excel 导出）
- 构建改为单文件 `--onefile` 模式，`dist/` 下仅输出一个 `.exe`
- 保存参数文件名加入当前时分秒（`参数_YYYYMMDD_HHMMSS.json`）

- 修复"恢复默认值"后输入框不更新（版本化 widget key 机制）
- 修复"加载参数"导入后数据不刷新（递增 widget_ver）
- 修复 `FamilyFinance` 构造时引用未定义变量（移至 return 前）
- 修复 `启动.bat` 中文字符乱码（ANSI/GBK 编码 + 纯 ASCII 重写）
- 修复按钮 key 冲突导致的 `UnboundLocalError`

## v1.0

- **日期**：2026-06-07
- **作者**：Sunnytao

### 主要功能

- 评估对象支持个人/夫妻两种模式
- 7 类资产、3 类负债、4 类年收入、月消费输入
- 小孩教育 4 阶段（学前/小学/中学/大学），按年龄自动匹配费用
- 意外预留金（一次性扣除）
- 通胀率与资产收益率可配置（默认均为 2%）
- 逐年现金流模拟直至 85 岁，判断是否实现财务自由
- 净资产趋势折线图、年度收支柱状图、年度明细数据表
- 参数保存/加载/下载/上传（JSON），结果导出（CSV/Excel）
- 参数自动持久化到 `~\.finfreedom_config.json`
- 恢复默认值按钮
- PyInstaller onedir 打包为独立 Windows 程序
- 所有金额前端以万元显示，内部以元计算

### 已知限制

- 不含税收计算
- 不含社保/养老金
- 不含多情景对比

### 技术栈

Python 3.10+ / Streamlit 1.58 / Pandas / Plotly / PyInstaller
